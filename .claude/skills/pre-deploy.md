# pre-deploy

デプロイ前に必要な3つのチェックを順次実行し、結果をサマリー表示する。

## 手順

1. **Lint チェック**: `pnpm lint` を実行
2. **型チェック**: `npx tsc --noEmit` を実行
3. **プロダクションビルド**: `pnpm build` を実行
4. 各ステップの成否をまとめて報告する

## 報告フォーマット

全ステップ完了後、以下の形式で報告:

```
## デプロイ前チェック結果

- Lint:  ✅ OK / ❌ エラー N 件
- 型:    ✅ OK / ❌ エラー N 件
- Build: ✅ OK / ❌ 失敗

（エラーがあれば詳細と修正提案を記載）
```

## ルール

- エラーがあっても3ステップすべて実行する（途中で止めない）
- エラーが見つかった場合は修正提案を添える
- 自動修正はしない（報告のみ）
